1 Model

Model()

A full Ribasim model schematisation with all input.

Ribasim model containing the location of the nodes, the edges between the nodes, and the node parametrization.

1.1 Parameters

Name Type Description Default
modelname str Model name, used in TOML and GeoPackage file name. required
node Node The ID, type and geometry of each node. required
edge Edge How the nodes are connected. required
basin Basin The waterbodies. required
fractional_flow typing.Optional[ribasim.node_types.fractional_flow.FractionalFlow] Split flows into fractions. required
level_boundary typing.Optional[ribasim.node_types.level_boundary.LevelBoundary] Boundary condition specifying the water level. required
flow_boundary typing.Optional[ribasim.node_types.flow_boundary.FlowBoundary] Boundary conditions specifying the flow. required
linear_resistance Linear flow resistance. required
manning_resistance typing.Optional[ribasim.node_types.manning_resistance.ManningResistance] Flow resistance based on the Manning formula. required
tabulated_rating_curve typing.Optional[ribasim.node_types.tabulated_rating_curve.TabulatedRatingCurve] Tabulated rating curve describing flow based on the upstream water level. required
pump typing.Optional[ribasim.node_types.pump.Pump] Prescribed flow rate from one basin to the other. required
terminal typing.Optional[ribasim.node_types.terminal.Terminal] Water sink without state or properties. required
discrete_control typing.Optional[ribasim.node_types.discrete_control.DiscreteControl] Discrete control logic. required
pid_control typing.Optional[ribasim.node_types.pid_control.PidControl] PID controller attempting to set the level of a basin to a desired value using a pump/weir. required
starttime typing.Union[str, datetime.datetime] Starting time of the simulation. required
endtime typing.Union[str, datetime.datetime] End time of the simulation. required
solver typing.Optional[ribasim.model.Solver] Solver settings. required

1.2.12 write

Model.write(self, directory)

Write the contents of the model to a GeoPackage and a TOML configuration file.

If directory does not exist, it is created before writing. The GeoPackage and TOML file will be called {modelname}.gpkg and {modelname}.toml respectively.

Name Type Description Default
directory FilePath required
